
## Installation
1. Clone Repo
    ```console
    git clone https://github.com/zhilanazmi/laravel-pemweb.git
    ```

2. Run
   ```console
   composer install
   ```
   ```console
   npm run install
   ```
   ```console
   cp .env.example .env 
   ```
   ```console
   php artisan key:generate
   ```
   ```console
   php artisan migrate
   ```
   ```console
   php artisan db:seed
   ```
   ```console
   npm run dev
   ```

3. Run server
   ```console
   php artisan serve
   ```
