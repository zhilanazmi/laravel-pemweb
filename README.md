### Installation

To install the application, follow these steps:

1. Clone the repository:

```console
git clone https://github.com/zhilanazmi/laravel-pemweb.git
```

2. Run the following commands:

```console
composer install
```

```console
npm run install
```

```console
cp .env.example .env
```

```console
php artisan key:generate
```

```console
php artisan migrate
```

```console

```

```console
npm run dev
```

3. Start the server:

```console
php artisan serve
```
